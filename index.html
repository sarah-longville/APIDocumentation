Developer Documentation

Learn how to use the Student API

What is the API about?

The purpose of this API is to provide a central point to store student information. Data can be also be stored regarding the courses and modules they are a part of. New data can be added, viewed, edited and deleted.

Getting started

This developer documentation will offer guidance as to how to effectivetly use this API. There are some code sample available which can be used to send requrest and details of getting connected with other developers.
You will need to have an API console in order to send requests. This isn't one available here but there are several online. An example is POSTMAN.
