Error Codes

HTTP Status codes

Code Text Description
304 Not modified No data has been changed in the request
400 Bad request The request entered is not valid. Please try another request.
401 Unauthorised Please ensure you have entered the correct authorisation credentials.
403 Forbidden You do not have the rights to carry out this request
404 Not found The data requested does not exist in the database. Make sure you are using the correct URI
500 Code Error Something has gone wrong with the internal code. Please visit the Twitter page to get help with this.
